Por onde começar com HTML e CSS

A Web é a plataforma de programação mais importante do mundo, e é praticamente onipresente. Suas tecnologias principais, o HTML e o CSS, são fundamentais para a criação de qualquer site.

Quem domina essas tecnologias, suas boas práticas e seus recursos mais avançados, tem espaço no mercado de programação para front-end e pode participar da criação de plataformas incríveis.

A Formação HTML e CSS (nome do curso) foi pensada para você começar do zero e se tornar um profissional de ponta. É um guia de estudos com um passo a passo pensado com carinho pela equipe

Os cursos dessa categoria ensinam, além do HTML e CSS, as bibliotecas e ferramentas mais importantes como o Bootstrap, CSS Grid e Devtools. Conheça a Arquitetura CSS, aprofunde-se na Acessibilidade de um site e entenda o protocolo da web: o HTTP.
